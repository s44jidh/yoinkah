# YOINKAH! 🚀
A minimalist, high-performance 4K media downloader with a sleek capsule UI.

Yoinkah! is a simple video and audio downloader that can download mp3/mp4 from various sites like YouTube and more. This was developed with python and is based on YTDLP and FFMPEG.

## ✨ Features
* **4K UHD Support:** Download video up to 2160p resolution.
* **Animated Capsule UI:** Real-time progress tracking built into the main "Yoink" button.
* **Smart Muxing:** Automatically merges high-quality video and audio streams using FFmpeg.
* **Dual Themes:** Instant toggle between **Dark Mode** 🌙 and **Light Mode** ☀️.
* **Smart Tagging:** Automatically formats filenames with resolution tags (e.g., `Video [1080p].mp4`).
